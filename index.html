hexo 搭建静态博客－基础篇

博客这种比较古老的方式，虽然现在没有以前流行了，但是它的很多优点却有着不可替代的作用。 hexo，一个静态的博客生成系统， 和 github，一个免费的托管网站， 使用他们，你就能搭建属于自己的博客啦！

鉴于 hexo 比较偏技术化，所以我不推荐一般人使用 hexo。 其它的博客托管服务，如新浪博客等，可能更适合你。 但是对于对 IT 感兴趣或者本身就是 IT 的，想搭建博客强烈建议 hexo！ 你不仅能建立一个漂亮的博客，还能从中学习到一些东西。

本文介绍了一些搭建 hexo 静态博客的准备工作和基本知识，力求通俗易懂，给想搭建博客的人一个参考。

博客开张

中秋前后，利用 hexo 工具和 github，终于把自己的博客给搭建起来了。 时间过的可真快，不知不觉的已经踏入大学三年了。 其实早在大二开始的时候，我就有搭建博客的想法了。 可那时，我发现我身上还有很多的问题要去解决。 而且我自己又太懒，所以这事，就一直拖着了。

前几天在整理自己的笔记，突然发现多了许多灵感。 于是就借着这波劲，把自己的博客给搭建起来了。 以后这里对我而言，是一个静谧之地，能让我暂时远离尘世的喧嚣和浮躁，直面内心的自己。 我不仅能够安静的深入思考问题，整理思绪，还可以在这里分享一些个人的小心得和浅薄的见解。

感谢这些年来一直默默支持我的朋友们，祝愿我们的明天将会更美好。

这是一篇使用 org-mode 书写的博文

org-mode 是 emacs 中用来编辑文档的主模式。 org-mode 使用自己的一套语法来标记富文档，类似 markdown，但是语法不同而且功能更多。 org-mode 与 emacs 集成的天衣无缝，能使用 emacs 的编辑优势专心写作，流畅编辑，不受打扰。 因此，拿 org-mode 写博客也是很自然的想法。

这篇博客是一篇测试博客，测试了在我的 hexo 配置中，最终对 org-mode 的展示效果如何。 :grinning:
